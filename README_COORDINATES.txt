Как расставлять точки на своей SVG-карте

1. Запусти проект через локальный сервер, например:
   python -m http.server 8080

2. Открой в браузере:
   http://localhost:8080/point-picker.html

3. Кликни по нужному месту на карте.
   Координаты автоматически появятся вверху и скопируются в буфер обмена:
   "x": 2430.4, "y": 871.4

4. Открой data/projects.json и замени x/y у нужного проекта.

Текущая карта:
viewBox="0 0 4127.1641 2050.417"

То есть корректный диапазон координат:
x: 0 - 4127.1641
y: 0 - 2050.417

В архиве также оставлен файл:
data/projects.original-coordinates.json

Это резервная копия старых координат до пересчёта под новую SVG-карту.
